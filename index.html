About Me
Hi! I’m Gabriel Ganzer (he/him/his), a computer engineer from Campinas, Brazil. Currently, I hold the position of Embedded Software Developer at Kryptus EED S.A. developing secure software for integrated avionic systems. I’m also experienced with firmware development for System-on-Chip (SoC) architectures and energy-efficiency hardware design and verification.

Projects

Deluxe Microprocessor

DLX is a fully pipelined RISC processor based on the Harvard Architecture designed for energy-efficiency and testing purposes, with a custom Instruction Set Architecture.

RTSNoC Sniffer

Design and evaluation of a non-intrusive packet delivery monitoring service on a Network-on-Chip (NoC) that focus on real-time Systems-on-Chip (SoC) running multimedia applications with variable-bit-rate.

Power Optimization

Synopsys® PrimeTime™ command in TCL that implements a post-synthesis power minimization procedure. The new command runs a leakage-constrained dual-Vth cell assignment with gate re-sizing.

Vaccine Refrigerator

VARE is a low-cost vaccine refrigeration system that can be used to adapt a domestic fridge into a specialized medical one.

Tunable Lamp

Smart Home system to control room lighting implemented using a S32K144 evaluation board running the Micrium µC/OS3 embedded operating system.

Dynamic Power Management

Dynamic Power Management (DPM) of a series of worloaks by simulating and modifying a Power State Machine (PSM) written in C language.

Energy Efficient Image Processing

Evaluation of several image transformations used in Organic Light-Emitting Diode (OLED) displays as a mean of reducing power consumption. Compensation techniques are also used in conjunction with Dynamic Voltage Scaling (DVS) for the same goal.

IoT Energy Management

Simulation of an IoT device in MATLAB/Simulink. The model consisted of a System-on-Chip (SoC) for measuring urban polution, integrating an the following sensors: air quality, methane level, temperature, and noise.

EZGas

EZGas is a crowdsourcing service, supported by a web application accessible both via smartphone or PC, that allows users to collect prices of fuels in different gas stations, as well as to locate gas stations.
